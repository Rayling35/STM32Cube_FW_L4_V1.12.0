STM32CubeL4 Firmware Package V1.12.0 / 22-May-2018

Main Changes
  • Add support of HAL callback registration/deregistration feature to overwrite HAL weak callbacks by user application functions
  • Maintenance release of HAL, Low Layer drivers and package examples.
  • Add FreeRTOS_MPU application projects for NUCLEO-L476RG and STM32L4R9I-EVAL boards.
